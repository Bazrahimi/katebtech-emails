## Development checks

```

pnpm typecheck
pnpm build

```

## How to use this framework in a website repo

### Remove old package

```bash
pnpm remove @katebtech/framework
```

### Install in a website project

```bash
pnpm add @katebtech/emails@git+ssh://git@github.com/Bazrahimi/katebtech-emails.git#main

```
